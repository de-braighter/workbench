# Context Minimizer

## What this skill does
- Build a minimal working set of files and interfaces
- Prefer diffs over full files
- Create a compact state summary
- Output a strict context pack for the next step

## Output contract
1) Task intent
2) Constraints
3) Working set include and exclude
4) State summary
5) Payload (diffs or snippets)
