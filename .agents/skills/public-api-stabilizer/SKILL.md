# Public API Stabilizer

## What this skill does
- Audit current Inputs and Outputs
- Propose a stable ViewModel contract and domain events
- Provide a migration plan if changes are breaking

## Output contract
1) API audit
2) Target contract
3) Migration plan
4) Patch guidance
