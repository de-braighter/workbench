# Angular Feature Generator (Decomposition Aware)

## What this skill does
- Propose Nx libs and tags (feature, ui, data-access)
- Produce a slice structure: page, container, facade, presentational, optional sections
- Define ViewModel-based presentational APIs

## Output contract
1) Proposed libs and tags
2) Folder tree and file list
3) Responsibilities and APIs
4) Next steps
