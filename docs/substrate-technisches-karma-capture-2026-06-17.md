1. Es gibt nicht „die" buddhistische Wahrnehmungstheorie

Es gibt mehrere Schichten:

| Schicht | Inhalt |
|---|---|
| Frühe Lehrreden / Nikāyas | sechs Sinnesbasen, Kontakt, Gefühl, Wahrnehmung, Begehren |
| Abhidhamma | extrem feine Prozessanalyse von Bewusstseinsmomenten, citta, cetasika, rūpa |
| Madhyamaka / Nāgārjuna | Wahrnehmungsobjekte haben keine Eigenexistenz; sie entstehen abhängig und konventionell |
| Yogācāra | Wahrnehmung ist stark bewusstseinsabhängig; Subjekt-Objekt-Dualität wird als konstruiert analysiert |
| Dignāga / Dharmakīrti | Wahrnehmung vs. begriffliches Denken; pratyakṣa und anumāna, direkte Wahrnehmung und Schlussfolgerung |

Das ist nicht ein System, sondern eine historische Evolution. Aber der gemeinsame Kern ist ziemlich stabil: Wahrnehmung ist bedingt, zusammengesetzt, instabil und anfällig für Projektion.

2. Der frühe buddhistische Kern: sechs Sinnesbasen

Der Buddhismus denkt Wahrnehmung nicht mit fünf Sinnen, sondern mit sechs:

| Sinn | Objekt |
|---|---|
| Auge | Formen / Sichtbares |
| Ohr | Klänge |
| Nase | Gerüche |
| Zunge | Geschmäcker |
| Körper | Berührbares |
| Geist | Gedanken, Vorstellungen, Erinnerungen, mentale Objekte |

Das ist schon genial, weil Gedanken nicht als „privater innerer Besitzer" behandelt werden, sondern als Sinnesobjekte. Der Geist nimmt Gedanken wahr wie das Auge Formen wahrnimmt. In MN 148 werden die sechs inneren Sinnesfelder, sechs äußeren Sinnesfelder, sechs Arten von Bewusstsein, sechs Kontakte und sechs Gefühle systematisch entfaltet; dort heißt es auch, dass Bewusstsein abhängig von Sinnesbasis und Objekt entsteht und dass das Zusammentreffen dieser drei Kontakt ist.

In Substrate-Sprache:

| Buddhistisch | Substrate |

| Sense base | Sensor / Interface / Adapter |
| Sense object | beobachtbares Objekt / Signal |
| Consciousness | domänenspezifischer Wahrnehmungsprozess |
| Contact | Observation Event |
| Feeling | Signal Valence |

Der entscheidende Satz:

Wahrnehmung beginnt nicht beim Subjekt. Wahrnehmung beginnt beim Kontakt.
