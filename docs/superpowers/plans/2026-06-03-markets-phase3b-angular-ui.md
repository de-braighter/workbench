# Markets Phase 3b — Angular Readout UI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build `apps/markets-ui/` — an Angular 17+ standalone app on port 4300 that polls `GET /readout` every 10 seconds and renders three asset cards (BTC/ETH/ADA) showing the Normal-Normal posterior mean (geometric mean price), 80% CI, source health, and confidence, using de-braighter glass panel design tokens.

**Architecture:** Single `AppComponent` as the shell — polls the API on an `interval(10_000)` observable and writes results to a signal. An `AssetCardComponent` accepts one `AssetReadout` as an `@Input()` and renders the glass panel. A `ReadoutService` wraps `HttpClient`. Design tokens are copied from `layers/design-system` and imported in `styles.css`. `proxy.conf.json` routes `/api` to `:3300`; CORS is enabled in the NestJS API.

**Precondition:** Phase 3a merged — `GET /readout` returning posteriors, `POST /ingest` working, DB running on :5455.

**Tech Stack:** Angular 17+ (standalone, signal-based), Angular CLI, `@angular/common/http`, TypeScript 5.4, port 4300.

**Spec:** `docs/superpowers/specs/2026-06-03-markets-phase3-inference-readout-design.md` §§6–8

**Repo:** `D:/development/projects/de-braighter/domains/markets/` (Phase 3a merged to main)

```typescript
import { Component, input, computed } from '@angular/core';
import { CommonModule } from '@angular/common';
import type { AssetReadout, SourceHealthStatus } from '../readout.types';

const TICKER: Record<string, string> = {
  bitcoin:  'BTC',
  ethereum: 'ETH',
  cardano:  'ADA',
};

const ASSET_NAME: Record<string, string> = {
  bitcoin:  'Bitcoin',
  ethereum: 'Ethereum',
  cardano:  'Cardano',
};

@Component({
  selector: 'app-asset-card',
  standalone: true,
  imports: [CommonModule],
  templateUrl: './asset-card.component.html',
  styleUrl: './asset-card.component.css',
})
export class AssetCardComponent {
  asset = input.required<AssetReadout>();

  ticker = computed(() => TICKER[this.asset().assetId] ?? this.asset().assetId.toUpperCase());
  name   = computed(() => ASSET_NAME[this.asset().assetId] ?? this.asset().assetId);

  /** Format a USD price with commas and appropriate decimal places. */
  formatPrice(usd: number): string {
    if (usd >= 1) {
      return new Intl.NumberFormat('en-US', { style: 'currency', currency: 'USD', maximumFractionDigits: 0 }).format(usd);
    }
    return new Intl.NumberFormat('en-US', { style: 'currency', currency: 'USD', minimumFractionDigits: 4, maximumFractionDigits: 6 }).format(usd);
  }

  healthClass(h: SourceHealthStatus): string {
    return `health-${h}`;
  }

  confidencePercent = computed(() => Math.round(this.asset().confidence * 100));
}
```

`apps/markets-ui/src/app/app.component.ts`:
```typescript
import { Component, inject, signal, OnInit, OnDestroy } from '@angular/core';
import { CommonModule } from '@angular/common';
import { interval, Subscription, startWith, switchMap } from 'rxjs';
import { ReadoutService } from './readout.service';
import { AssetCardComponent } from './asset-card/asset-card.component';
import type { ReadoutResult } from './readout.types';

@Component({
  selector: 'app-root',
  standalone: true,
  imports: [CommonModule, AssetCardComponent],
  templateUrl: './app.component.html',
  styleUrl: './app.component.css',
})
export class AppComponent implements OnInit, OnDestroy {
  private service = inject(ReadoutService);

  readout  = signal<ReadoutResult | null>(null);
  loading  = signal(true);
  error    = signal<string | null>(null);
  ingesting = signal(false);

  private sub?: Subscription;

  ngOnInit(): void {
    this.sub = interval(10_000).pipe(
      startWith(0),
      switchMap(() => this.service.readout()),
    ).subscribe({
      next: result => {
        this.readout.set(result);
        this.loading.set(false);
        this.error.set(null);
      },
      error: err => {
        this.loading.set(false);
        this.error.set(String(err));
      },
    });
  }

  ngOnDestroy(): void {
    this.sub?.unsubscribe();
  }

  triggerIngest(): void {
    this.ingesting.set(true);
    this.service.triggerIngest().subscribe({
      next: () => this.ingesting.set(false),
      error: () => this.ingesting.set(false),
    });
  }
}
```
